The Problem

Educat is a system based on the computerized adaptive test (CAT), which is a digital form of evaluation where the test is not defined a priori, it is constructed as the student is answering the questions.

The Solution

Although I found many systems based on the traditional learning method on my benchmarking research, the user research revealed that our clients' teachers had a different process to create their tests: they only created the items (or questions), leaving other professionals to evaluate and combine them. They had strict rules for its creation, following specialists reviews and classifying then accordingly to CAT methodology. The interface solution was to make an item repository with different permissions for different professionals. The item construction page has a review assistant (blue button and popover windows). The item listings show a status for each item, where the user can see if it has been approved or not.
